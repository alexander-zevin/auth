Для старта сайта необходимо запустить файл start.exe в папке bin и перейти по адресу: http://localhost:8181/. 
При написании проекта использовались следующие технологии: Go, SqlLite, HTML, CSS, JavaScript. 
А также сторонные библиотеки для Go: github.com/mattn/go-sqlite3 (драйвер для базы данных), github.com/dgrijalva/jwt-go (Токены).
